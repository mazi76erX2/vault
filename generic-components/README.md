# HC Generic Components
## Clone the repo

To clone this repo locally simple cd to your chosen folder and run:

```shell
    git clone https://dev.highcoordination.de/bitbucket/scm/tcplus/generic-components.git
```

## Run project

CD to your local repo, then install `node_modules`

```shell
    # npm 
    npm install
    
    # Or yarn
    yarn
```

After you have installed your npm packages, you can now run the following command:

```shell
    # npm run
    npm run storybook
    
    # Or yarn run
    yarn storybook
```

Then your storybook docs will be available at http://localhost:6006
