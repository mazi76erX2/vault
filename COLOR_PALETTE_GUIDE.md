## ðŸŽ¯ Color Usage

### Primary Colors

**Light Mode - Deep Blue (#6061c0)**

- Main buttons
- Links
- Active states
- Primary headings
- Important elements

**Dark Mode - Teal (#55BBAD)**

- Main buttons
- Links
- Active states
- Primary headings
- Important elements

### Accent Colors

**Error/Warning (#81334b light, #DD5794 dark)**

- Error messages
- Warning alerts
- Delete actions
- Important negatives

**Success/Positive (#6ea861 light, #55BBAD dark)**

- Success messages
- Checkmarks
- Completed states
- Positive indicators

**Info/Highlight (#ee9e6e light, #DBC1CA dark)**

- Information alerts
- Highlights
- New features
- Attention grabbers
